영어 대소문자와 띄어쓰기만으로 이루어진 문자열이 주어진다. 이 문자열에는 몇 개의 단어가 있을까? 이를 구하는 프로그램을 작성하시오. 단, 한 단어가 여러 번 등장하면 등장한 횟수만큼 모두 세어야 한다.



> 처음에 매우 쉽게 생각하고, 단순히 공백의 갯수만 세려고 했더니
>
> 공백으로 시작하는 경우, 공백으로 끝나는 경우가 생겨 조건문을 활용해야했다.
>
> 그럼에도 여러 오답이 나옴. 문제의 난이도에 비하면 생각보다 헤맸다... ^^

 

```python
arr = list(input())
cnt = 0
isEmpty = True
for i in range(len(arr)):
    if arr[i].isalpha():
        isEmpty = False
    if arr[i] == ' ':
        if i == 0 or i == len(arr) - 1:
            pass
        else:
            cnt += 1
if isEmpty:
    ans = cnt
else:
    ans = cnt + 1
print(ans)
```

> 기본적으로 공백으로 시작하거나 끝나는 값은 제외하였다.
>
> ` `a` ` 이와 같이 공백에 둘러싸인 경우를 생각해 기본값을 0으로 주고 ans 에서만 +1 해주는 방법을 넣어줬었따.
>
> 근데 계속 오답이 나와서 보니까 반례에 ` ` <- 아예 공백인 경우가 있었던 것. 공백이 연속으로 나오지 않는다는 조건에만 집중해서 문자가 나오지않을 거란건 생각하지 못했다.
>
> 이를 검증하기 위해서 isEmpty 변수를 넣어서, 문자가 나오면 t / f 값을 변동하도록 해, 문자가 끝까지 나오지 않으면 0 그대로, 한번이라도 나왔다면 + 1 해주도록하니 정답나옴
>
> 기본 문젠데 왤케 정답률 낮은 가 했더니 이런 사소한 케이스를 놓쳐서 그런거같다.
